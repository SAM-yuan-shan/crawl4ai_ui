## 🛠️ 安装和使用

### 快速启动（Windows）
1. 双击 `启动界面.bat`
2. 等待依赖检查完成
3. 界面自动打开

### 通用启动方法
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动界面
python start_ui.py

# 或直接运行
python crawl4ai_ui.py
```

### 环境测试
```bash
# 检查环境是否就绪
python test_environment.py
```
